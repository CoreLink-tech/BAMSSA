BAMSSA DELSU Chapter Static Website
===================================

This folder contains a static HTML/CSS/JS version of the BAMSSA DELSU Chapter website.

How to use:
- Open index.html in your browser to view the homepage.
- Other pages: about.html, departments.html, gallery.html, staff.html, events.html, news.html, contact.html, executives.html
- Styling comes from `style.css` and `site.js`, with Tailwind loaded from the CDN in each page.
- Local images are stored in `assets/` as optimized webp files, including the cleaned faculty logo.
- The original logo source is `logo.jpg`; the site now uses `assets/logo-clean.webp`.

You can replace the demo webp images in `assets/` with your own photos later if you want the site to show real campus and event images.

To deploy, upload the contents of this folder to any static web host (e.g. Netlify, Vercel, GitHub Pages, etc).

All mobile app code and dependencies have been removed. This is a pure static website.
